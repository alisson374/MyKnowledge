## Intro
### Busca
-> Solução como caminho em espaços de estados;
-> Sem interações com outros agentes
-> Agente com total controle sobre suas ações e efeitos

### Jogo
-> Existe um oponente imprevisível
-> Considerar também as ações do oponente

### O que é um jogo
-> Um ***Jogo*** é um modelo teórico de conflitos de interesse entre dois ou mais jogadores.
-> Um ***Jogador*** é um participante ativo e racional, cujos movimentos afetam os demais.
-> Os ***Movimentos*** são decisões disponíveis aos jogadores (simultâneos ou alternados):
	-> Composto por ações
	-> Baseado em ganhos (outcome, payoff, utility)
	-> Podendo ser um evento probabilístico

### Jogo - Definição Formal
***Estados*** 
	-> Conjunto de estados S, estado inicial S0

***Jogadores*** 
	-> Jogador(s): qual jogador se move no estado s

***Ações*** 
	-> Ações(s): conjunto de movimentos válidos no estado s
	-> Resultado(s,a): modelo de transição resultante

***Teste final***
	-> Teste de Término(s): determina quando o jogo termina
 
***Função de utilidade (Payoff )***
	-> Utilidade (s,p): valor numérico final no estado s para o jogador p
