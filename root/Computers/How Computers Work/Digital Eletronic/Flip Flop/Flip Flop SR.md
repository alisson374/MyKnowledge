### Transições

| Resumo | S   | R   |
| ------ | --- | --- |
| Zero   | 0   | X   |
| Um     | x   | 0   |
| Alfa   | 1   | 0   |
| Beta   | 0   | 1   |
*Zero*  -> mantém apagado
*Um*   -> mantém aceso
*Alfa*  -> acende
*Beta* -> apaga
