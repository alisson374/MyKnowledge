# Configuração das Portas IO do microcontrolador

* Portas Bidirecionais, permitindo a configuração de entrada ou saída;
* A instrução TRIS x Configura uma porta em função do código;
* 1 IN / 0 OUT
* Entradas são bits que efetuam leituras externas;
* Saídas são bits que acionam algum dispositivo interno;
