## Erro absoluto
A diferença entre o valor exato ( x ) e o valor aproximado ( x' ) no
cálculo computacional pode ser medida pelo erro absoluto
O erro absoluto ***Ea*** é definido como:
$$ E_a = |x - x'| $$

## Erro Relativo
A diferença entre o módulo do valor exato ( x ) e o valor
aproximado ( ) e o quociente pelo valor exato, no cálculo
computacional pode ser medida pelo erro relativo.
O erro relativo ER é definido como:

$$ E_r = \frac{|x - x'|}{|x|} ~~~~ ou  ~~~~  E_r =\frac{E_a}{|x|}$$
